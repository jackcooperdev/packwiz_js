# packwiz_js
> NodeJS controller for packwiz

packwiz is a command line tool for creating Minecraft modpacks. Instead of managing JAR files directly, packwiz creates TOML metadata files. packwiz_js allows for packwiz to be used in a Node.JS project.


### Setup

Download packwiz [here](https://packwiz.infra.link/) and either place in system path or take note of the path. Then install the package with the command below.

```
npm i @jackcooper04/packwiz_js
```
> This package is available on both the NPM registry and the Github Packages Registry.
